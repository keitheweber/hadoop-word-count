# Map Reduce - Word Count

Basic implementation of the "Word Count" example MapReduce program.

Sample input and output @ 
data/input 
data/output

# Compile

```sh
$ mvn clean install
```

# Run
```sh
$ ./hadoop jar /path/to/wordcount-1.0-SNAPSHOT.jar /path/to/inputDir /pathToOutputDir
```
